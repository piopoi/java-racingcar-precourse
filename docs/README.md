# 자동차 경주 게임
## 필요 기능
* 자동차
  * 위치 값을 가진다.
  * 전진 또는 정지 할 수 있다.
    * 0~9의 random값을 구한 후 4 이상이면 전진, 3이하면 멈춘다.
  * 이름을 가진다.
    * 이름은 5자 이하만 가능하다.
* 입력 기능
  * 자동차 이름 입력
    * 이름은 쉼표로 구분
  * 시도할 횟수 입력
    * 숫자만 입력
* 출력 기능
  * 각 차수별 실행 결과 출력
  * 게임 결과 출력
* 경주 결과
  * 결과 판정
    * 위치 값을 비교하여 우승자 판별.
* 경주 게임 컨트롤러
